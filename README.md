# dev

Personal dev configuration for setting up a new machine.

## Setup

### 1. Fonts

Install the fonts in the `fonts/` folder before anything else, as they are used by the terminal and other tools (this is done automatically on NixOS).

- **macOS**: Select all `.ttf` files in `fonts/FiraCode/`, right-click → "Open With → Font Book" (or double-click each and click "Install Font")
- **Windows**: Select all `.ttf` files in `fonts/FiraCode/`, right-click → "Install" (or "Install for all users")
